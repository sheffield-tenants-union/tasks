# Sheffield Tenants' Union Task Tracker

This is a task tracker 

This is link to list of all the tasks (github calls them "issues"): https://github.com/sheffield-tenants-union/tasks/issues

To add to the task list you will need to:
1) make a github account (give it email, password, verify this, etc).
2) click on the "New Issue" button and fill it out.
